# BeyondData Intellab

> **Empowering Africa through digital intelligence, mentorship & innovation.**

The BeyondData Intellab is a mentorship-driven ecosystem by [BeyondData ICT](https://beyonddataict.com) that equips young Africans with practical digital skills, career guidance, innovation exposure, employability readiness, and technology leadership opportunities for the digital economy.

---

## About the Project

This repository contains the website for the **BeyondData Intellab** platform — a future-focused mentorship, innovation, and employability hub designed to bridge Africa's growing digital skills and opportunity gap.

The platform connects aspiring learners, young professionals, entrepreneurs, and emerging innovators with experienced mentors, industry networks, and collaborative innovation spaces.

---

## Features

- **Hero section** — Program introduction with cohort announcements and CTA buttons
- **Core program overview** — 8-week Digital Career Accelerator breakdown
- **Curriculum pillars** — 5 training modules covering digital foundations, data literacy, marketing, AI, and career strategy
- **Who we serve** — Four target tracks with gender inclusion commitment
- **Cohort system** — Structured intake with timelines, capacity info, and selection process
- **Application flow** — 3-step application, screening, and selection process
- **Mentorship section** — For both mentors and mentees
- **Opportunities** — Internships, volunteer pathways, and partner connections
- **Impact goals** — Long-term metrics and community growth targets
- **FAQ** — Accordion-style frequently asked questions
- **Testimonials** — Community voices and reviews
- **Footer** — Links, contact info, social media, and newsletter signup

## Program Structure

The BeyondData Intellab runs in **structured cohorts** of 50 mentees and 10 mentors over 8 weeks:

| Week | Focus |
|------|-------|
| 1 | Orientation & digital foundations |
| 2 | Data & digital literacy |
| 3 | AI, automation & modern work tools |
| 4 | Personal branding & visibility |
| 5 | Career & employability readiness |
| 6 | Entrepreneurship & innovation |
| 7 | Project development & portfolio building |
| 8 | Showcase, assessments & pathways forward |

---

## Curriculum Pillars

1. **Digital Foundation & Productivity** — Google Workspace, professional communication, digital organisation
2. **Data Literacy & Analysis** — Data basics, Excel, dashboards, interpreting insights
3. **Digital Marketing & Visibility** — Personal branding, LinkedIn optimisation, portfolio development
4. **AI & Automation** — Intro to AI models, responsible AI use, workflow automation
5. **Career & Income Strategy** — Digital career pathways, remote work, freelancing, business digitisation

## Getting Started

### Prerequisites

//- Node.js >= 18
- npm or yarn//

### Installation

```bash
# Clone the repository
git clone https://github.com/your-org/beyonddata-intellab.git

# Navigate into the project
cd beyonddata-intellab

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for production

```bash
npm run build
npm run start
```

## Design Reference

The website follows a clean, trust-building design system:

- **Primary colour** — `#185FA5` (BeyondData blue)
- **Dark headings** — `#0C447C`
- **Accent / light** — `#B5D4F4`
- **Dark section bg** — `#1A2A5E` (navy, used for "Who We Serve")
- **Success / teal** — `#0F6E56`
- **Typography** — System sans-serif stack, weights 400 and 500 only

Wireframes and design assets are maintained separately. Contact the design team for Figma access.
